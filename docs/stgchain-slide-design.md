# Visual Slide Design - STG-Chain

## 1. Tema Visual
- **Warna Utama**: Biru futuristik (#1E3A8A) + aksen silver (#C0C0C0).
- **Background**: Grid blockchain dengan nuansa kosmik.
- **Tipografi**: Sans-serif modern (Montserrat / Open Sans).
- **Ikonografi**: Hexagonal untuk STG‑Chain, lingkaran glowing untuk reputasi.

---

## 2. Layout Konsisten
- **Header**: Logo STG‑Chain + tagline "Sovereign Titan Genesis".
- **Body**: Konten utama dengan bullet ringkas atau grafik.
- **Footer**: Hashtag #STGChain #ReputationOnChain #Web3.

---

## 3. Desain Per Slide
- **Slide Judul**: Logo besar, tagline, latar glowing biru‑silver.
- **Slide Visi**: Ilustrasi lingkaran reputasi di tengah, teks visi di sekeliling.
- **Slide Value Proposition**: 5 ikon (transparansi, kepercayaan, meritokrasi, interoperabilitas, DApp).
- **Slide Roadmap**: Timeline horizontal dengan milestone 2026–2028.
- **Slide Traction**: Grafik pertumbuhan komunitas + angka kunci.
- **Slide Kesimpulan**: Poster reputasi dengan tagline ikonik.

---

## 4. Visualisasi Tambahan
- Gunakan **infografis reputasi** sebagai elemen grafis utama.
- Tambahkan **poster ekosistem** di slide roadmap.
- Pastikan konsistensi warna dan ikon di seluruh deck.
