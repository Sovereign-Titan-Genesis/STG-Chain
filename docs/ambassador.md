# STG-Chain Ambassador Program (Draft)

## Tujuan
Membangun komunitas global STG-Chain dengan individu yang berperan sebagai jembatan antara tim inti dan komunitas lokal.

---

## 1. Peran Ambassador
- **Promosi**: memperkenalkan STG-Chain ke komunitas lokal.
- **Edukasi**: menyelenggarakan workshop, webinar, dan tutorial.
- **Komunitas**: membangun grup Telegram/Discord lokal.
- **Feedback**: mengumpulkan masukan dari komunitas untuk tim inti.

---

## 2. Insentif
- Reward STG token untuk aktivitas komunitas.
- Badge reputasi khusus di chain.
- Akses eksklusif ke testnet dan program developer.
- Kesempatan menjadi validator komunitas.

---

## 3. Roadmap
- **Batch 1**: 10 ambassador di Asia Tenggara.
- **Batch 2**: ekspansi ke Eropa & Amerika.
- **Batch 3**: jaringan global dengan 100+ ambassador.
