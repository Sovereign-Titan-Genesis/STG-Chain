# Dashboard Analitik STG-Chain

## 1. Tampilan Utama
- **Grafik Engagement Rate**: line chart per platform (Twitter, LinkedIn, Telegram).
- **Grafik Reach & Impressions**: bar chart bulanan.
- **Growth Komunitas**: area chart jumlah followers/subscribers.
- **Conversion Metrics**: funnel chart dari engagement → registrasi SOUL‑ID → staking → DAO voting.
- **Sentiment Analysis**: pie chart komentar positif vs negatif.

---

## 2. Fitur Dashboard
- **Real‑Time Update**: data otomatis sinkron dari API Twitter, LinkedIn, Telegram.
- **Filter Waktu**: harian, mingguan, bulanan.
- **Filter Platform**: pilih per channel atau gabungan.
- **Export Data**: CSV/Excel untuk laporan investor.

---

## 3. KPI yang Ditampilkan
- Engagement Rate (%)
- Reach & Impressions (jumlah audiens unik)
- Growth Komunitas (followers/subscribers baru)
- Conversion Metrics (registrasi SOUL‑ID, staking, DAO voting)
- Sentiment Analysis (% positif vs negatif)
- Content Performance (poster, infografis, thread edukasi)

---

## 4. Visualisasi
- Warna biru‑silver futuristik sesuai branding STG‑Chain.
- Ikon hexagonal untuk setiap KPI.
- Latar belakang grid blockchain untuk nuansa Web3.
